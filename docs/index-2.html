2. Installation

2.1 Where to get MyAddressbook

MyAddressbook can be downloaded from:

  • http://rootus.org/src.php - as a gzipped source-tarball

2.2 Requirements

For MyAddressbook to work you need web server with PHP enabled and MySQL database.

Required:

  • MySQL http://www.mysql.com
  • PHP http://www.php.net

2.3 Installation

2.3.1 Linux

Go to the web server folder and unpack the tar.gz file there.

$ cd /var/www/html/
$ tar xfvz MyAddressbook_1.1.0.tar.gz

This will create an folder called "addressbook".

Go to the "docs" folder where the Database layout file is

$ cd addressbook/docs

Create an empty database called MyAddressbook.

$ mysqladmin -u username -ppassword create MyAddressbook

Create the database tables using the "MyAddressbook.sql"

$ mysql -u username -ppassword MyAddressbook < MyAddressbook.sql

Edit the file "db_config.php" to set database
variables (hostname , username , password)

2.3.2 Windows

Using winzip or something similar unpack the tar.gz file to the WWW folder.

In this example I had the WWW document root in "D:\www\Apache2\htdocs"
and MySQL was installed at "d:\www\mysql"

D:\>cd www\Apache2\htdocs
D:\www\Apache2\htdocs>cd addressbook
D:\www\Apache2\htdocs\addressbook>d:\www\mysql\bin\mysqladmin.exe create
MyAddressbook
D:\www\Apache2\htdocs\addressbook>cd docs
D:\www\Apache2\htdocs\addressbook\docs>d:\www\mysql\bin\mysql -u username
-ppassword MyAddressbook < MyAddressbook.sql

Using Notapade I then edit the file "db_config.php" to set database
variables (hostname , username , password)

That's it...now you should be able to launch you web browser and start using
this addressbook.

2.4 Upgrade

To upgrade from 0.7 and older you need to add the config database table.
This table can be found in the "docs" directory.

Earlier versions you only need to overwrite all the .php files,
the database layout has not changed so there are no more steps required.
