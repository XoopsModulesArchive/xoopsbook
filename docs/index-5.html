5. Changelog

1.1.1

  • Updated Dutch language file from Peter Smessaert
  • Added Brazilian Portuguese lanuage file from Rogerio Acquadro
  • Fixed bug that did not allow underscore or minus in E-Mail addresses.
  • Added Windows installation to the Manual.
  • Some minor fixes.

1.1.0

  • Auto set focus on the Search box.
  • Added French translation by bruno@peronne.org.
  • Fixed bugs regarding missing DB variables, if used different database table
    names Admin page did not work.

1.0.9

  • Some HTML fixes.
  • Updated the Manual.
  • Added mapquest.com support.

1.0.8

  • Added Swedish translation.
  • Save directly in Outlook as vCard, using PHP class by kaib@bitfolge.de.
  • Export now sends the file for Mozilla and OE to your computer, makes
    things a lot simpler.
  • Some addins for the Report Generator.
  • Started to write this Manual
  • Minor layout fixes.

1.0.7

  • Cleanup, now all variables you need to change are in db_config.php file.
  • Renamed some files to make more sense.
  • Added option to hide "Add New" and disable "Delete" and "Update", set in
    the Admin page (Read-Only Mode).
  • Lots of small bugfixes...eg. language files updates, missing tags fixes....

1.0.6

  • Added Report Generator, where you can print out the entire addressbook
    using selected fields.

1.0.5

  • Moved the language select to the index page, should not have to login to
    change it
  • Some small fixes for the language change

1.0.4

  • Added German translation from Jens (jens@woodpeckerzone.de).
  • Fixed the "Change Language" mess, dropped all these silly symlinks and now
    it is simply session based.
  • E-Mail address validation.

1.0.3

  • MyAddressbook will now work with "register_globals = Off" in php.ini

1.0.2

  • Changed the Admin page...alot easier to navigate now.
  • Started work on the Import function, but as mysql "load data infile" is no
    longer an option it could get messy.
  • Dropped all the icon theme stuff....was fun to make but it doesnt make any
    sense.
  • Deleted print.php....its now a function, dont need a page for that.
  • Extended the setup.sh script to create and import the initial database, if
    wanted.

1.0.1

  • Added Export feature, saves the database as an text file that can be
    Imported to either Mozilla or Outlook Express.
  • Several bugs reported by Peter Smessaert fixed (I hope).
  • Added a REALLY basic password protection on the Admin page, just trying to
    keep this simple.
  • Added a check for the language file, forgot that the last time and I guess
    everybody had problems woth that version.
  • Updated the README file.

1.0.0

  • Clean up, clean up and more clean up.
  • Fixed fonts to look better in Mozilla under Linux.

0.9

  • Added "Order by" Name, Address or E-Mail on listview.
  • Added limit 15 on listview and "Next" to view more
  • Added a "Print Preview".
  • 0.8 had the wrong .sql file to create new database, fixed.
  • Few fixes on the table mess.

0.8

  • Added an Admin page, where you can change layout and colors.
  • Changed the dbconnect function to have password by default, forgot it last
    time.
  • Changed the Lang setup and added a easy way to change it via the new Admin
    page (mind you, I guess it only works in Linux).
  • Added support for iconthemes, you can choose between GNOME and MS icons on
    the Admin page.
  • Made a setup shellscript (UNIX only) to fix MySQL username and password.
  • Minor fixes.

0.7

  • Added a Dutch translation from Peter Smessaert (Peter@Smessaert.be).
  • Fixed a bug that made "Add new" not working after doing "Show all".
  • Some minor fixes.
